## 두 수의 곱

---

### - 문제

정수 num1, num2가 매개변수 주어집니다. num1과 num2를 곱한 값을 return 하도록 solution 함수를 완성해주세요.

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = 0;
        return answer;
    }
}
```

### - 출력 예시

```
num1	num2	result
3	    4	    12
27	    19	    513
```

<!-- >  -->

---

### - 답

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = 0;
        answer = num1 * num2; // 반환하기 위해 answer에 num1과 num2의 곱을 넣어줌
        return answer;
    }
}
```
