## 짝수와 홀수

---

### - 문제

정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.

```java
class Solution {
    public String solution(int num) {
        String answer = "";
        return answer;
    }
}
```

### - 출력 예시

```
num	return
3	"Odd"
4	"Even"
```

<!-- >  -->

---

### - 답

```java
class Solution {
    public String solution(int num) {
        String answer = "";
        if (num % 2 == 0) { // 2로 나눴을때 나머지가 0 이면 짝수
            answer = "Even";
        } else {
            answer = "Odd";
        }
        return answer;
    }
}
```
