## 두수의 차

---

### - 문제

정수 num1과 num2가 주어질 때, num1에서 num2를 뺀 값을 return하도록 soltuion 함수를 완성해주세요.

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = 0;
        return answer;
    }
}
```

### - 출력 예시

```
num1	num2	result
2	    3	    -1
100	    2	    98
```

---

### - 답

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = 0;
        answer = num1 - num2; // num1값에서 num2값을 뺀것을 answer에 넣어줌.
        return answer;
    }
}
```
