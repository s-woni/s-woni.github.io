![run3-1](/assets/img/blog/sql-run/run3-1.png)

```sql
SELECT u.user_id, u.email, COALESCE(p.point, 0) FROM users u  
LEFT JOIN point_users p ON u.user_id = p.user_id  
ORDER BY p.point DESC;
```
> **COALESCE(A, 10)** => A가 NULL 값인 경우 10으로 대체
