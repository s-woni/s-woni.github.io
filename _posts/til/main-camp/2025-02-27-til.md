## 📖 Today I Learned
### Lv 1. 클래스 없이 기본적인 연산을 수행할 수 있는 계산기 만들기

<!-- <h4> 📃  </h4> -->

---

#### - 전체 코드

```java
package calc.level1;

import java.util.Scanner;

public class App {
    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        while (true) {
            System.out.println();
            System.out.println("계산기를 시작합니다 (exit 입력 시 종료)");

            System.out.println("-----------------------------------------");

            int firstNum = 0;
            int secondNum = 0;
            char calculation;

            while (true) {
                System.out.print("첫 번째 숫자를 입력해주세요 : ");

                if (scanner.hasNext("exit")) {
                    System.out.println();
                    System.out.println("계산기를 종료합니다.");
                    scanner.close();
                    return;
                }
                if (scanner.hasNextInt()) {
                    firstNum = scanner.nextInt();
                    if (firstNum < 0) {
                        System.out.println();
                        System.out.println("음수를 입력하셨습니다.");
                        System.out.println("양수를 입력해주세요.");
                        System.out.println();
                        continue;
                    }
                    scanner.nextLine();
                    break;
                } else {
                    System.out.println();
                    System.out.println("숫자를 입력해주세요.");
                    System.out.println();
                    scanner.nextLine();
                }
            }

            while (true) {
                System.out.print("두 번째 숫자를 입력해주세요 : ");

                if (scanner.hasNext("exit")) {
                    System.out.println();
                    System.out.println("계산기를 종료합니다.");
                    scanner.close();
                    return;
                }
                if (scanner.hasNextInt()) {
                    secondNum = scanner.nextInt();
                    if (secondNum < 0) {
                        System.out.println();
                        System.out.println("음수를 입력하셨습니다.");
                        System.out.println("양수를 입력해주세요.");
                        System.out.println();
                        continue;
                    }
                    scanner.nextLine();
                    break;
                } else {
                    System.out.println();
                    System.out.println("숫자를 입력해주세요.");
                    System.out.println();
                    scanner.nextLine();
                }
            }

            while (true) {
                System.out.print("사칙 연산 기호를 입력하세요 (+, -, *, /) : ");

                if (scanner.hasNext("exit")) {
                    System.out.println();
                    System.out.println("계산기를 종료합니다.");
                    scanner.close();
                    return;
                }

                calculation = scanner.next().charAt(0);

                if ("+-*/".indexOf(calculation) != -1) {
                    if (secondNum == 0 && calculation == '/') {
                        System.out.println();
                        System.out.println("나눗셈 연산에서 두번째 정수에 0이 입력될 수 없습니다.");
                        System.out.println();
                        continue;
                    }
                    break;
                } else {
                    System.out.println();
                    System.out.println("잘못된 입력입니다.");
                    System.out.println("+, -, *, / 중 하나를 입력해주세요.");
                    System.out.println();
                }
            }

            int result = 0;

            switch (calculation) {
                case '+':
                    result = firstNum + secondNum;
                    break;
                case '-':
                    result = firstNum - secondNum;
                    break;
                case '*':
                    result = firstNum * secondNum;
                    break;
                case '/':
                    result = firstNum / secondNum;
                    break;
            }
            System.out.println("-----------------------------------------");

            System.out.println("결과 : " + firstNum + " " + calculation + " " + secondNum + " = " + result);
        }
    }
}
```

![main-til-02-27-1](/assets/img/blog/main-til/main-til-02-27-1.PNG)  
- 위 사진과 같이 첫 번째 숫자에 숫자가 아닌 다른 값을 입력하면 무한히 반복되는 에러 발생
- 원인
  - 스캐너 버퍼 안에 남아있던 값으로 인해 `while`과 그 안에 있는 `else`가 무한히 반복됨
- 해결
  - `scanner.nextLine();`을 사용하여 스캐너 버퍼 안에 남아있던 값을 비워줌

```java
while (true) {
  System.out.print("첫 번째 숫자를 입력해주세요 : ");

  if (scanner.hasNext("exit")) {
    System.out.println();
    System.out.println("계산기를 종료합니다.");
    scanner.close();
    return;
  }
  if (scanner.hasNextInt()) {
    firstNum = scanner.nextInt();
    if (firstNum < 0) {
      System.out.println();
      System.out.println("음수를 입력하셨습니다.");
      System.out.println("양수를 입력해주세요.");
      System.out.println();
      continue;
    }
    break;
  } else {
    System.out.println();
    System.out.println("숫자를 입력해주세요.");
    System.out.println();
    scanner.nextLine(); // 스캐너 버퍼 안에 남아있던 값을 지워줌
  }
}
```
- 마무리
  - 찾아보았더니 스캐너 버퍼와 관련된 문제는 꽤나 자주 발생하는 이슈인 것 같다. 특히 `nextInt()`와 같은 메서드는 숫자만 읽고 개행문자는 버퍼에 남아 있어 그 후에 입력을 제대로 처리하지 못하는 경우가 많다. 이러한 문제를 `nextLine()`을 사용하면 버퍼를 쉽고 효과적으로 해결해준다는 것을 알게 되었다.

![main-til-02-27-2](/assets/img/blog/main-til/main-til-02-27-2.PNG)  
- 무한히 반복 되는 것을 해결 했으나 두번째 계산 즉, 두번째 입력부터 값이 두번 출력 되는 현상이 있었음
- 원인
  - 위에서 해결했던 문제와 같이 스캐너 버퍼안에 남겨진 값으로 인한 문제
- 해결
  - 이 또한 `scanner.nextLine()`을 사용하여 간단하게 해결할 수 있었음

```java
while (true) {
  System.out.print("첫 번째 숫자를 입력해주세요 : ");

  if (scanner.hasNext("exit")) {
    System.out.println();
    System.out.println("계산기를 종료합니다.");
    scanner.close();
    return;
  }
  if (scanner.hasNextInt()) {
    firstNum = scanner.nextInt();
    if (firstNum < 0) {
      System.out.println();
      System.out.println("음수를 입력하셨습니다.");
      System.out.println("양수를 입력해주세요.");
      System.out.println();
      continue;
    }
    scanner.nextLine(); // 스캐너 버퍼 안에 남아있던 값을 지워줌
    break;
  } else {
    System.out.println();
    System.out.println("숫자를 입력해주세요.");
    System.out.println();
    scanner.nextLine(); // 스캐너 버퍼 안에 남아있던 값을 지워줌
  }
}
```
